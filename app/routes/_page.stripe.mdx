# Stripe

Stripe uses advanced machine learnig algotirhms to detect scams and abuse. To prevent rejection, make sure that:

1. LLC with matching EIN
2. Use a real US address, not registered agent's PO box or mail forwarding.
   Works if:

   - Dedicated suite number (not shared)
   - Real commercial address (not a PO box or RA office)
   - Ability to forward mail and provide proof of address (PDF scan of utility bill or lease)
     To check:

3. IP and browser stack match physical addr
4. US phone number and bank account (Mercury default choice)
5. Biz website looking legit
